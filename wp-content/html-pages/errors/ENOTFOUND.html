wp-now: Error starting: getaddrinfo ENOTFOUND # Error: getaddrinfo ENOTFOUND This error occurs when the server is unable to resolve the hostname. This can happen for a variety of reasons, such as a misconfigured DNS server, a typo in the hostname, or a network issue. In the context of wp-now, this error can occur when trying to start the server with a custom URL that is not properly configured.
